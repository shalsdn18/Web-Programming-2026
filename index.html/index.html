웹의 역사와 창시자 팀 버너스리

팀 버너스리 소개

팀 버너스리(Tim Berners-Lee)는 1955년 6월 8일 영국에서 태어난 컴퓨터 과학자이자 물리학자로, 월드 와이드 웹(WWW)의 발명자로 널리 알려져 있습니다. 그는 1989년에 CERN에서 일하면서 정보 공유를 위한 시스템을 고안했고, 이를 바탕으로 1990년에 첫 번째 웹 브라우저와 서버를 개발했습니다. 팀 버너스리는 웹의 창시자로서, 인터넷을 통해 정보를 쉽게 공유할 수 있게 만든 공로로 2004년 튜링상을 수상했습니다. 또한, 그는 W3C(World Wide Web Consortium)를 창립하여 웹 표준의 발전에 기여하고 있습니다.

웹의 역사

웹은 1980년대 말에 시작되어 빠르게 발전해 왔습니다. 아래 테이블은 웹의 주요 발전 단계를 연대별로 정리한 것입니다.

연대 주요 특징
1989 팀 버너스리가 WWW의 개념을 제안하고, 하이퍼텍스트 시스템을 고안
1990 첫 번째 웹 브라우저(월드와이드웹)와 웹 서버 개발
1991 첫 번째 웹사이트(info.cern.ch) 공개 및 HTTP 프로토콜 발표
1993 모자이크 브라우저 출시, 그래픽 인터페이스 도입으로 웹 대중화 시작
1994 W3C(World Wide Web Consortium) 창립, 웹 표준화 작업 시작
1995 넷스케이프 내비게이터 출시, 자바스크립트 지원으로 동적 웹 시작
1998 XML 1.0 표준화, 데이터 교환을 위한 마크업 언어 등장
2000년대 초반 웹 2.0 개념 등장, 블로그, 위키, 소셜 미디어의 시작
2004 팀 버너스리 튜링상 수상
2010년대 HTML5와 CSS3 표준화, 모바일 웹의 급속한 발전
2020년대 웹 3.0, 블록체인과 AI 통합, 개인정보 보호 강화
